⚙ 管理員一次性設定指南（業務忽略此區）

⚠ 此區僅 Project 管理員會用到。業務正常使用本頁不需動 Claude.ai Project 設定。

① 建立 Project

Claude.ai → Projects → New project → 命名「ARTWAY 提案生成」

② Files：上傳 Project 知識

  • 業務提案簡報規格.md ── 視覺規範（顏色 / 字型 / 版型語法）
  • template.html ── 視覺參考實作（AI 看不抄整份，可從中學習風格）
  • 服務目錄（從 Notion 同步，方法見 ③）

註：html-rules_editor.md 是編輯器規格，AI 生成的 artifact 不含編輯器（自由模式），所以不用上傳。

③ 服務目錄同步（二選一）

  • 方法 A（推薦）：Claude.ai 個人設定 → Connectors → 連結 Notion ── 之後 Claude 在 Project 對話時可自動讀
  • 方法 B：手動把 Notion 服務目錄複製貼到 .md 檔，當第 4 個 Files 上傳，更新時手動重傳

④ Instructions：把 Instructions.md 內容貼到 Project Instructions 欄位

開啟同資料夾的 Instructions.md → 全選複製 → 貼到 Project Instructions（在 Project 右上角）。

未來想調整 AI 行為（例如新增提案結構規則）只動 Instructions.md，改完重貼到 Project Instructions 即可。

⑤ 完成後

業務 workflow：

  1. 開本頁 brief-builder.html → 填表 → 複製 brief
  2. 到 Claude.ai 「ARTWAY 提案生成」Project → 開新對話 → 貼上 brief → 送出
  3. 等 1-3 分鐘 → AI 回 HTML artifact（含 print CSS）
  4. 下載 artifact 為 .html → 雙擊打開 → 瀏覽器 Ctrl+P → 另存為 PDF

如果想用編輯器（點字直接改、AI 重生這頁等）做進一步調整，可開 template.html 手動將 slide 內容複製過去。
